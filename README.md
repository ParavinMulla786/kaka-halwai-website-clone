## 📌 Overview
This project is a **responsive front-end clone** of the Kaka Halwai website, developed using **HTML, CSS, and Bootstrap**.

It replicates the design and layout of a traditional sweets brand website while maintaining **modern UI/UX practices** and full **mobile responsiveness**.

---

## 🚀 Features
- 📱 Fully responsive design (Mobile, Tablet, Desktop)
- 🎨 Clean and structured UI
- 🧭 Navigation bar with smooth layout
- 🍩 Product showcase sections
- 📩 Contact form UI
- ⚡ Optimized static performance

---

## 🛠️ Tech Stack
- **HTML5** – Structure  
- **CSS3** – Styling  
- **Bootstrap 5** – Responsive layout & components
